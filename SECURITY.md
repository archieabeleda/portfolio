# Security Policy

## Reporting a Vulnerability

If you find a security issue in this repository or on https://archieabeleda.dev,
please report it privately. Do not open a public issue for security problems.

Email: hello@archieabeleda.dev

Please include:

- A description of the issue and where you found it
- Steps to reproduce, if you have them
- Any relevant logs or screenshots

I aim to acknowledge reports within a few days and will keep you posted on the fix.
Thanks for helping keep this project secure.
